# ClipboardCenter
A fast and convenient clipboard
一个快捷方便的剪贴板应用
你是否有过为了复制粘贴文本而在各个窗口穿梭的苦恼？
那么，我开发这款软件的初衷就是为了这个！

此项目在[码云](http://git.oschina.net/mrams/ClipboardCenter)上
